<h1 align="center">Assets</h1>

This repository contains all assets regarding marketing, branding and theming of EL & E InfiNet SolutionsPH OPC.

## Directory Structure

```bash
.
├── assets                      # All images used for marketing, branding, and theming.
│   ├── favicon                 # Circular logos for websites.
│   ├── logo                    # Main logo and design.
│   └── typography              # Word version logo for InfiNet.
└── src                         # Source / project files of the images (.psd).
```
